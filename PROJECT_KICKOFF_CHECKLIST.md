Project Kickoff Checklist

This repository follows the suggested kickoff checklist for research-grade reproducible projects.

High-level items:
- Version control & branching: main + dev + feature branches (feat/...)
- Monorepo layout: frontend/, backend/, docs/, simulations/
- ADRs in docs/decisions/
- API contract first: .env.example provided
- Dev infra: docker-compose.yml (local dev)
- Docs split by audiences (Users / Developers / Reviewers)

See the repo files for templates and examples.
